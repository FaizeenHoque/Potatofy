A fun Chrome/Arc extension that replaces every image on a webpage with a potato. Works in real time, even with lazy-loaded images.

## Features
- Replace all images with potatoes
- Works on scroll and dynamically loaded content
- Random potato images for variety

## Installation
1. Go to `chrome://extensions`
2. Enable Developer Mode
3. Load unpacked folder containing this extension

Enjoy your potato-powered browsing!
